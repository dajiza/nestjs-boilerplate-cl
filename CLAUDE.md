# Boulevard Sync - 已同步表格

> **警告：以下表格字段已与 Boulevard API 对齐，禁止随意更改字段名和类型。**

> 新增/修改同步表格的 entity 时，字段名、类型、可空性必须参照 `boulevard/SCHEMA_REFERENCE.md` 和 `boulevard/schema.json`。

已同步的 10 张表：

- `business` — 商户信息
- `location` — 门店位置
- `service_category` — 服务分类
- `service` — 服务项目
- `staff` — 员工
- `staff_role` — 员工角色
- `client` — 客户
- `appointment` — 预约
- `shift` — 班次
- `timeblock` — 时间块
